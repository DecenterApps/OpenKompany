# OpenKompany


### Preqrequisites
You need to have node.js and webpack installed on your machine.

To use the app, you need to have MetaMask installed and point it to the Rinkeby testnet.
 
 
## Buidling locally

```
yarn
yarn dev
- OR -
npm install 
npm run dev
```

## Building a production bundle

```
yarn
yarn prod
- OR -
npm install 
npm run prod

```
